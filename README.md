# Fashion Management

A fashion management application built with Java and CSS.

## 🛠️ Technologies

- **Java** (88%) - Primary programming language
- **CSS** (12%) - User interface styling

## 📋 Description

This project provides tools to manage fashion collections, track trends, and organize outfits.

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or higher
- Maven or Gradle package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MrBuddhaCoder/quanlythoitrang.git
cd quanlythoitrang
```

2. Build the project:
```bash
mvn clean install
```

3. Run the application:
```bash
mvn spring-boot:run
```
